# CorpSim v0.5.0 开发计划

## 📊 当前状态 (v0.4.1)

### ✅ 已完成
- [x] Railway部署成功 - https://openclawgame-production.up.railway.app/
- [x] 基础游戏框架 (3人董事会模式)
- [x] API端点 (创建会话、加入游戏、发送消息、投票)

4. **投票机制简单** - 缺乏议程绑定投票
